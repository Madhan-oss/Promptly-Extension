🛠️ Setup Guide

Install in 30 seconds

Choose your browser below and follow the simple steps to get Promptly running.

1

Download the ZIP

Click the Chrome download button below to download promptly-chrome-v1.2.0.zip to your computer.

⬇ Download for Chrome
2

Unzip the File

Right-click the downloaded ZIP file and select "Extract All" (Windows) or double-click (Mac) to unzip it into a folder.

3

Open Chrome Extensions

In Chrome, go to chrome://extensions in the address bar and press Enter.

4

Enable Developer Mode

Toggle "Developer mode" ON using the switch in the top-right corner of the Extensions page.

5

Load the Extension

Click "Load unpacked", then select the folder you unzipped in Step 2. Promptly will appear in your extension list! ⚡

1

Download the Firefox ZIP

Download promptly-firefox-v1.2.0.zip — this is the special Firefox MV2 build.

⬇ Download for Firefox
2

Open Firefox Debugging

In Firefox, type about:debugging in the address bar and press Enter.

3

Click "This Firefox"

In the left sidebar click "This Firefox" to open the local add-on loader.

4

Load Temporary Add-on

Click "Load Temporary Add-on…", then navigate into the unzipped folder and select manifest.json. Done! ⚡

1

Download the ZIP
